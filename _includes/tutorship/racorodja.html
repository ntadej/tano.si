Python

Namestitev
  • Windows: prenesi z python.org, ob namestitvi obkljukaj "Add Python to PATH" oz. pod "Advanced" "Set Python environment variables"

  • Linux: namesti paket svoje distribucije (običajno python3)
  • jupyter/ipython: za namestitev uporabi pip3 install jupyter ipykernel
